# Configuration des équipements Cisco

## Présentation

Ce dossier regroupe l'ensemble des configurations réalisées au cours de ce laboratoire.

L'objectif est de documenter de manière claire et structurée la configuration des différents équipements Cisco utilisés dans cette architecture réseau.

Les fichiers présents permettent de comprendre la mise en œuvre des principaux mécanismes d'administration réseau tout en constituant une base réutilisable pour de futurs projets.

---

# Contenu

Les configurations comprennent notamment :

- Configuration initiale des équipements
- Attribution des noms d'hôtes
- Sécurisation des accès
- Configuration des interfaces
- Plan d'adressage IPv4
- VLAN
- Routage inter-VLAN
- DHCP
- DNS
- NAT / PAT
- ACL
- SSH
- EtherChannel (LACP)
- HSRP
- OSPF
- Routes statiques
- Sauvegarde des configurations

---

# Objectifs pédagogiques

À travers ces configurations, plusieurs compétences ont été développées :

- Comprendre la syntaxe IOS Cisco
- Administrer un commutateur Cisco
- Administrer un routeur Cisco
- Configurer un réseau local professionnel
- Déployer des services réseau
- Sécuriser les accès aux équipements
- Mettre en œuvre des protocoles de routage
- Concevoir une architecture évolutive

---

# Organisation du dossier

Chaque fichier correspond à un équipement ou à une fonctionnalité particulière afin de faciliter la lecture des configurations.

Les configurations sont volontairement conservées séparées afin de permettre leur réutilisation dans d'autres laboratoires.
