###############################################################################
# Dépannage
###############################################################################

Connexion refusée :

- Vérifier que le service ProFTPD est démarré.
- Vérifier que le port 21 est ouvert.

Impossible de joindre le serveur :

- Vérifier l'adresse IP.
- Vérifier la connectivité réseau.

Erreur d'authentification :

- Vérifier que les connexions anonymes sont autorisées
  dans le fichier proftpd.conf.

Répertoire vide :

- Vérifier que le chemin /srv/ftp est correctement configuré.
- Vérifier la présence des fichiers de test.
